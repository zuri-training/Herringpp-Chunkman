Getting Started

Inorder to gain access to the platform, you need to be registered as a user. As a registered user, you will get to enjoy the following benefits:

  • A personal dashboard
  • Full access to the platform
  • Access to CSV and JSON format
  • Access to chunk and save files
  • Access to download files after chunking
  • Access to other features that may be added in the nearest future

Unregistered users also have access to the following features

  • Access to visit the platform and view basic information
  • Access to Documentation page, Landing page, About us page and the contact section
  • Access to register, become a member and enjoy all the benefits of the site

About the platform

Technology used:

  • Frontend: HTML, CSS and javascript was used for building of the website
  • Backend: Nodejs was used to authenticate users, uploading files, chunking, saving, validating, downloading and checking for progress of downloading
  • Database Schema: Database schema is the "Blueprint" of a database that describes how thw data may relate to other tables or other data models. An express was used as the framework. MangoDB was used as the database
